### SpringELK

1) Запущен ELK (есть готовый docker-compose; нужно около 2гб ОЗУ)
2) Через интерфейс Kibana подключён к elasticsearch (первоначальная инициализация)
3) Тестовые логи записываются в файл app.log
4) Настроен logstash.conf для проброса логов в ELK, для каждого сервиса создан свой файл
5) Через Kibana можно сделать запрос ранее записанных логов, а также их фильтровать, искать и т.п.
